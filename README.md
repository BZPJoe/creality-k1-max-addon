# Creality K1 Max Add-ons Repository

This repository contains Home Assistant add-ons for Creality K1 Max 3D printer integration.

## Available Add-ons

### Creality K1 Max Monitor

Monitor and control your Creality K1 Max 3D printer from Home Assistant.

**Features:**
- Real-time printer status monitoring
- Temperature monitoring (bed and extruder)
- Print progress tracking
- MQTT integration for Home Assistant
- Home Assistant MQTT Discovery support
- Supports Moonraker API (Klipper firmware)

## Installation

1. Add this repository to your Home Assistant Add-on Store:
   - Go to **Settings** → **Add-ons** → **Add-on Store**
   - Click the three dots (⋮) → **Repositories**
   - Add: `https://github.com/BZPJoe/creality-k1-max-addon`

2. Install the add-on from the Add-on Store

3. Configure the add-on with your printer's IP address and MQTT settings

4. Start the add-on
